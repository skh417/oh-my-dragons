# Oh My Dragons

**드래곤 타마고치** - 브라우저에서 플레이하는 픽셀 스타일 드래곤 육성 게임

## 게임 소개

알에서 드래곤을 부화시키고, 먹이를 주고, 놀아주며 성체까지 키워보세요!
어떤 타입의 드래곤이 나올지는 랜덤입니다.

## 실행 방법

### 방법 1: 직접 열기
`index.html` 파일을 더블클릭하여 브라우저에서 열기

### 방법 2: 로컬 서버 사용
```bash
npx http-server -p 8080
```
브라우저에서 `http://localhost:8080` 접속

## 플레이 방법

### 1. 알 부화하기
- 알을 클릭하거나 케어 버튼을 눌러 부화시키세요
- 성장 게이지가 15 이상이 되면 랜덤 타입의 드래곤이 태어납니다

### 2. 드래곤 케어
| 버튼 | 효과 |
|------|------|
| 먹이주기 | 배고픔 +20, 성장 +3 |
| 놀아주기 | 행복 +25, 에너지 -15, 성장 +5 |
| 재우기 | 에너지 회복 (100까지), 성장 +2 |

### 3. 성장 단계
알 → 아기 (25%) → 청소년 (50%) → 성체 (100%)
